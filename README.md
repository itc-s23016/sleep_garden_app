# Sleep Garden

アラームと連動して育成が進む、睡眠習慣サポートアプリ。

## 機能概要
- アラーム設定 / ON-OFF
- 起床達成で育成（XP/成長）
- 図鑑（植物一覧・入手状況）
- データ保存（Room）

## 使用技術
- Kotlin
- Android Studio
- Jetpack (ViewModel / LiveData or StateFlow)
- Room (SQLite)

## 説明
-アラームを設定して
-眠るボタンを押して画面を落として眠りにつく
-五時間以上眠ると花GET
-獲得した花は図鑑ボタンから見ることができる
-注意寝るボタンを押さないと花がもらえないから注意
